## 🔧 Funcții JavaScript cheie (în index.html)

### toggleTheme()
- Toggle dark/light mode
- Setează `html.light-mode` și `body.light-mode`
- CSS variables: `--bg-primary`, `--bg-secondary`, `--bg-tertiary`, `--border-color`, `--text-primary`, `--text-secondary`, `--accent`
- Salvare preferință în localStorage

### toggleMenu()
- Deschide/inchide dropdown menu (buton hamburger stânga sus)
- Conține filtre surse + căutare

### filterNews(source, element)
- Filtrage știri după sursă (Digi24, Mediafax, Europa FM, Libertatea, Adevărul, G4Media)
- Adaugă clasa `active` pe butoane
- Afișează doar știrile cu sursa selectată

### searchNews(query)
- Căutare în titlu și conținut știri
- Apelat și pe Enter
- Afișează buton "×" pentru ștergere căutare

### resetFilters()
- Resetează toate filtrele și căutarea

### shareNews(newsItem, event)
- Click pe buton "share" → copiază URL în clipboard
- Prevent click event să deschidă știrea

### toggleWeatherForecast() / toggleCurrencyRates() / toggleCryptoRates()
- Expand/collapse pentru secțiuni de detalii
- Toggle display block/none
- Lazy loading: datele se preiau la prima deschidere

### showCityDropdown() / selectCity(name, lat, lon) / searchCity(query)
- Căutare oraș cu geocoding API (Open-Meteo Geocoding)
- Salvare oraș selectat în localStorage

## 🤖 Chatbot AI (chatbot.js)

### Arhitectură
- **IIFE pattern** — totul wrapped în `(function() { ... })();`
- Se inițializează când DOM e ready
- **Clasă**: `AziNewsChatbot`

### Configurație
```javascript
const WORKER_URL = 'https://azinews-chatbot.garconai93.workers.dev';
const SYSTEM_PROMPT = 'Ești un asistent AI pentru site-ul de știri românesc AziNews...';
```

### Funcționalități
1. **Chat button** — fixed bottom right, gradient albastru
2. **Chat panel** — se deschide când apeși butonul
3. **Welcome suggestions** — 4 butoane predefinite cu întrebări
4. **Message history** — se păstrează în conversație (nu persistă pe reload)
5. **Typing indicator** — animație bouncing dots
6. **Error handling** — toast roșu pentru erori
7. **Auto-resize** textarea

### Flux conversație
1. User trimite mesaj
2. Se showTyping()
3. Se apelează `fetch('all_news.json')` pentru context
4. Se trimite POST la Cloudflare Worker cu `message` + `context`
5. Worker-ul folosește AI (Hugging Face Inference API probabil)
6. Răspunsul se afișează ca bubble

### CSS Chat UI
- Culori: `#161b22` (dark), `#58a6ff` (accent blue)
- Font: Inter
- Responsive: sub 480px — panel full width

## 📰 update_news.py

### Surse RSS
```python
SOURCES = [
    ("Digi24", "https://www.digi24.ro/rss"),
    ("Mediafax", "https://www.mediafax.ro/feed"),
    ("Europa FM", "https://www.europafm.ro/feed/"),
    ("Libertatea", "https://www.libertatea.ro/rss"),
    ("Adevarul", "https://adevarul.ro/rss"),
    ("G4Media", "https://www.g4media.ro/feed"),
]
```

### Smart Shuffle Logic
```python
cutoff = now - timedelta(hours=4)  # 4+ hours = old

new_news = []  # < 4h de la fetch
old_news = []  # >= 4h

for item in all_news:
    fetched = datetime.fromisoformat(item['fetched_at'])
    if fetched >= cutoff:
        new_news.append(item)
    else:
        old_news.append(item)

random.shuffle(new_news)  # AMESTECĂ știrile noi
display_news = new_news + old_news  # Noi sus (amestecate), vechi dedesubt (ordine)
```

### Conținut știri
- **MAX 500 chars** — se taie la ultimul cuvânt complet (nu mid-word)
- Title max **150 chars**
- Image URL din `<enclosure>` sau `<media:content>`
- Parsare time/date din pubDate (RFC 2822)

### Deduplicare
- Compara URL-uri existente vs noi
- URL-uri noi adăugate la **începutul** listei

### Output
1. **all_news.json** — salvare integrală (persistă istoricul)
2. **index.html** — replace `const news = [...]` cu array-ul actualizat
3. Titlul paginii actualizat: `Ultimele știri din România • 06 Apr 14:51 (3262 știri)`

## 📝 Rețete rapide (pentru viitoare modificări)

### Adaugă o nouă sursă de știri
1. Adaugă tuple în `SOURCES` din `update_news.py`
2. RSS feed trebuie să aibă `<item>` elements cu `<title>`, `<link>`, `<pubDate>`, `<description>`

### Modifică chatbot prompt
1. Editează `SYSTEM_PROMPT` în `chatbot.js`

### Adaugă o nouă funcție de filtering
1. Adaugă element în `menu-options` div din `index.html`
2. Adaugă `onclick="filterNews('key', value, this)"`
3. Modifică `filterNews()` în JS

### Schimbă intervalul de update
1. Modifică cron expression în `.github/workflows/update-news.yml`
2. Format: `minut oră zi lună zi-săptămânână`

### Adaugă o nouă crypto în toggle
1. Adaugă ID în fetch URL CoinGecko
2. Adaugă icon în `const icons` object
3. Extinde `toggleCryptoRates()` function

## ⚠️ Lucruri de evitat

- **Nu modifica `const news = [...]` manual** — se suprascrie la fiecare run de update_news.py
- **all_news.json nu se editează manual** — e regenerat de script
- **Titlurile au max 150 chars, content max 500 chars** — taie automat
- **fetched_at** este ISO format pentru smart shuffle (4h cutoff)
